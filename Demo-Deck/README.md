# Demo card deck

This is a demo 58 card deck.
It contains:

- 52 poker cards.
- 2 jokers.
- 4 tichu cards.

It uses:

- Card Composer to generate the cards.
- The default Compose font for the numbers and suits.
- Material Icons.

To get a printable a version of this deck:
- Clone the repository.
- In this "Demo-Deck" folder, run the `./gradlew run` command.
- In the app, click on the export button (top right button).
